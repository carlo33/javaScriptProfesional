HERENCIA PROTOTIPAL

SomeObject.prototype es un objeto donde definimos el prototipo de las instancias de SomeObject. Es decir las instancias del SomeObject van a heredar del SomeObject.prototype

FUNCIONES NUEVAS

  1. hasOwnProperty("name")
    2. Para verificar si una propiedad le pertenece a un objeto
  2. Object.getPrototypeOf(zelda)
    3. Nos permite obtener el prototipo de un objeto

CONCLUSIONES

1. Al crear diferentes metodos dentro de una función estos se guardar en el PROTOTYPE, cuando instanciamos esta función a un nuevo objeto, estos metodos tambien se instancia dentro del PROTOTIPE del nuevo objeto, pero en realidad es el mismo objeto

2. El prototype del prototype es heredado del "Object" que es la base para crear funciones
